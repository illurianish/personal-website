
# Personal Website – Anish Reddy Illuri

This is my personal portfolio website built using HTML, CSS, and JavaScript.  
It features a responsive layout, embedded resume, GitHub/LinkedIn integration, and a modern professional design.

## 🌐 Live Site

[https://illurianish.github.io/personal-website/](https://illurianish.github.io/personal-website/)

## 🛠 Features

- Responsive layout (mobile-first)
- Clean sidebar with profile, contact info, and social links
- Projects section linked to GitHub
- Embedded resume viewer (PDF)
- Google Maps location (East Meadow, NY)
- Highlighted past experience logos (Sriven Info, Software Labs, Fordham)
- Contact section with large LinkedIn banner

## 🚀 Deployment

This website is hosted via GitHub Pages:
- Branch: `main`
- GitHub Pages Source: `/ (root)`

To set up a custom domain (like `illurianish.com`), update the repository's CNAME and configure DNS settings.
